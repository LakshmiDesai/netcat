1. Создать список - BannerAreas. Добавить туда все возможные области показа

2. Создать компонент.. Готовый лежит тут же.. NetCat_2011_class.xml
поля:
2331	Area	Область показа баннера						
2332	File	Файл баннера						
2333	Link	Ссылка для перехода						
2334	ViewCount	Число просмотров						


3. В месте показа баннера:
<?
$b = new pBanner(2011); // 2011 - ClassID компонента
echo $b->rotate(1); // 1 - область показа
?>

4. в файле go.php нужно заменить ClassID при создании класса pBanners


ВНИМАНИЕ! Компонент пока работает только на том уровне вложенности, как тут:
т.е. /netcat/modules/default/petun/banners
